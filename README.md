# Perfume Digital Clock

香水の調香と拡散をモチーフにした、単一HTMLファイル構成のデジタル時計です。  
現在時刻・日付・曜日を表示しながら、24時間それぞれに対応したテーマカラー、花、花言葉、Scent Profile が切り替わります。

昼は「香りを抽出してボトルに満たしていく香水」、夜は「香りがゆるやかに広がりながら残量が減っていくリードディフューザー」として振る舞うよう設計しており、同じ時計の中で時間帯ごとに異なる香りの物語を楽しめます。

---

## 特徴

- **HTML / CSS / JavaScript の単一ファイル構成**
- **現在時刻（時:分:秒）をリアルタイム表示**
- **日付（年/月/日）と曜日を表示**
- **24時間ごとにテーマカラー・花・花言葉・Scent Profile が切り替わる**
- **昼と夜で演出モードが切り替わる**
  - **昼（9:00〜20:59）**：香水の抽出・充填モード
  - **夜（21:00〜8:59）**：リードディフューザーの拡散・残量減少モード
- **PC / スマホ両対応のレスポンシブデザイン**
- **外部ライブラリ不使用**

---

## デザインコンセプト

この時計は、単なる時刻表示ではなく、  
**「時間帯ごとに異なる花の香りが、昼は抽出され、夜は静かに広がっていく」**  
という体験をテーマにしています。

- ページ全体の背景は `#ffe0c7` を基調にしつつ、夜帯では深く静かなトーンへ変化
- メインスクリーンは、その時間帯のテーマ色を反映したガラス調の背景
- Floral Note カードは、白系の半透明フロストガラス風
- カード内文字色は背景とのコントラストを意識し、明るい面ではブラウン系、暗い面では白系を採用
- 昼は「抽出されて満ちる」、夜は「拡散しながら減っていく」という逆方向の時間演出を持たせている

## 表示仕様

### 時計表示
- 表示形式：`HH:MM:SS`
- 24時間表記
- 秒までリアルタイム更新

### 日付表示
- 表示形式：`YYYY/MM/DD`
- 曜日は日本語表記
- 日付と曜日は横並び表示

### Floral Note 表示
各時間帯ごとに以下を表示します。

- 花の名前
- 花言葉
- テーマ色
- Scent Profile（どのような香りかの説明）

## 演出仕様

## 昼の演出（9:00〜20:59）

昼の時間帯は、**香料が雫として抽出され、ボトルに満ちていく香水演出** になります。

### 充填進行
- 1本のボトルは **10分で100%** になります
- 進行率は以下の通りです
  - **1% = 6秒**
  - **10分 = 600秒 = 100%**

### 抽出演出
- 上部の花モチーフから香りが抽出されるイメージ
- 雫が滴るアニメーション
- チューブを通ってボトルへ落ちる構成
- 液体がボトルの下から上へ満ちていく
- スクリーン背景には、時間帯に応じたやわらかな光感を持たせる

### 10分ごとの演出
- ボトルが 100% に達すると、次のボトルへ切り替わる
- その瞬間に、**花びらが画面全体を右から左へ流れる演出** が入る
- 花びらの色や雰囲気は、その時間帯の花テーマに連動

---

## 夜の演出（21:00〜8:59）

夜の時間帯は、**リードディフューザーとして香りがゆるやかに広がり、残量が減っていく演出** に切り替わります。

### 残量進行
- ボトル残量は **1時間で100%から0%へ減少** します
- その時間帯の香りが、1時間かけて周囲へ広がっていくイメージです

### ディフューザー演出
- ボトル上部にリードスティック（棒）を表示
- ボトル側面には、その時間帯の花モチーフを装飾として表示
- 香りが静かに広がるような、穏やかな拡散アニメーションを表示
- **夜帯では雫演出を行いません**
- **夜帯では背景のライト演出も行いません**
- 昼よりも静かで落ち着いた見た目にしています

### 夜の考え方
昼が「香りを作る時間」なら、夜は「香りが部屋に満ちて広がる時間」です。  
そのため、夜は抽出の動きを減らし、余韻と拡散に重心を置いています。

---

## 1時間切り替え演出

毎正時には、10分ごとの演出とは別に、  
**次の時間帯の花を強く印象づける花びら演出** が入ります。

- **左から右へ** 画面全体を花びらが流れる
- 通常の10分演出よりも強め
- 次の時間帯の色と花テーマへ移行する“場面転換”として機能する

## レスポンシブ仕様

## PC表示

- 想定サイズ：**約 900px × 400px**
- 左右2カラム構成
- 左側に時計・日付・Floral Note
- 右側にボトル / ディフューザー演出

### PCレイアウトの主な構成
- 左：eyebrow、時刻、日付、曜日、Floral Noteカード
- 右：花モチーフ、雫またはリード、ボトル本体、液面 / 残量表示

### PC時の見え方
- 時計と演出を同時に眺めやすい
- 香水ビジュアルと情報のバランスがよい
- デスクトップ壁紙や常時表示用途を意識した横長構成

## スマホ表示

- 1カラム構成へ切り替え
- **画面全体を香水瓶に見立てた見せ方** に変化
- 時計表示と情報カードを中央に寄せる
- 背景側にボトルの存在感を持たせる

### スマホ時の主な調整
- `eyebrow` を中央寄せ
- `time-wrap` を中央寄せ
- `info-panel` を高さ方向の中央付近に配置
- 下部には `safe area` を考慮した余白を確保
- スマホの操作UIと干渉しにくい設計

### スマホ時の見え方
- 時計が主役として中央に見える
- 背後で香りの演出が静かに進行しているように見える
- 画面そのものが香水瓶のラベル面のように感じられる構成

## 時間帯テーマについて

24時間それぞれに対して、以下の情報を定義しています。

- テーマ色
- 色のイメージ
- 花の名前
- 花言葉
- Scent Profile
- ボトル表示用の色
- 花モチーフ描画用の設定

これにより、**0時から23時までそれぞれ異なる香りと色彩の表情** を持たせています。

## カスタマイズしやすいポイント

### 1. 時間帯テーマの変更
JavaScript内のテーマ配列を書き換えることで、各時間帯の以下を変更できます。

- 花名
- 花言葉
- Scent Profile

カード背景やテーマ色に応じて、文字色の切り替えロジックを調整すると、より可読性を高められます。
