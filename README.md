# Perfume Digital Clock

香水の調香をモチーフにした、シングルファイル構成のデジタル時計です。  
現在時刻・日付・曜日を表示しながら、時間帯ごとにテーマカラーと花のモチーフが切り替わります。  
香料が少しずつ抽出されてボトルに満ちていくような演出と、満了時に花びらが流れるアニメーションを組み合わせ、視覚的にも香りの時間経過を感じられる時計として設計しています。

---

## 特徴

- **HTML / CSS / JavaScript の単一ファイル構成**
- **現在時刻（時:分:秒）をリアルタイム表示**
- **日付（年/月/日）と曜日を表示**
- **24時間ごとにテーマカラー・花・花言葉・香りの説明が切り替わる**
- **1本のボトルが10分で満ちる進行演出**
  - 1% = 6秒
  - 100% 到達時に次のボトルへ切り替わる
- **満了時に、画面全体を花びらが右から左へふわっと流れる演出**
- **香水の抽出をイメージした雫・チューブ・ミスト演出**
- **PC / スマホ両対応のレスポンシブデザイン**
- **スマホ表示時は画面全体を香水瓶に見立てた構成**

---

## デザインコンセプト

この時計は、単なる時刻表示ではなく、  
**「時間ごとに異なる花の香りを抽出してボトルに閉じ込める」** という体験をテーマにしています。

- ページ全体の背景は、基本色 `#ffe0c7` をベースにしつつ、夜の時間帯では暗く深いトーンへ遷移
- メインスクリーンは、その時間帯のテーマカラーを反映したガラス調の背景
- 花の情報カードは、**白を基調とした半透明のフロストガラス風**
- ボトル内の液体はテーマカラーに合わせて変化
- 満了時には、その時間帯の花をイメージした花びらが画面上を流れる

---

## 表示仕様

### 時計表示
- 表示形式：`HH:MM:SS`
- 24時間表記
- 秒までリアルタイム更新

### 日付表示
- 表示形式：`YYYY/MM/DD`
- 曜日は日本語表記

### 花の情報表示
各時間帯ごとに以下を表示します。

- 花の名前
- 花言葉
- テーマ色名
- Scent Profile（どのような香りかの説明）

---

## ボトル演出仕様

### 充填進行
- 1本のボトルは **10分で100%** になります
- 進行率は以下の通りです
  - **1% = 6秒**
  - **10分 = 600秒 = 100%**

### 抽出演出
- 上部の花モチーフから香りが抽出されるイメージ
- チューブ内を流れるストリーム
- 雫が落ちるアニメーション
- ボトル内の液体が下から上へ満ちていく

### 満了時演出
- 100% 到達時にボトルをリセット
- そのタイミングで、**画面全体に花びらが右から左へ流れる演出** を発生
- 花びらの色や雰囲気は、その時間帯の花テーマに連動

---

## レスポンシブ仕様

### PC表示
- 想定サイズ：**約 900px × 400px**
- 左側に情報エリア、右側に香水ボトル演出を配置
- 2カラム構成で、時計と香水演出を同時に見せるレイアウト

#### PCレイアウトの主な構成
- 左：タイトル、時刻、日付、曜日、Floral Noteカード
- 右：花のソースモチーフ、抽出チューブ、雫、ボトル、充填率

---

### スマホ表示
- 1カラム構成へ切り替え
- **画面全体を香水瓶に見立てた演出** に変化
- ボトル演出は背景側にまわり、時計表示と情報表示を中央に重ねる構成
- `eyebrow` と `time-wrap` は中央寄せ
- `info-panel` も縦方向中央寄せ
- 画面下部には、スマホの操作UIと干渉しにくいよう **safe area を考慮した余白** を確保

#### スマホ時の見え方
- 時計が主役として中央に見える
- 背後で香水が抽出されているように見える
- 花の説明カードは読みやすさを保ちつつ、半透明で軽やかに表示

## 実装構成

この作品は **単一の `.html` ファイル** で完結しています。

### 使用技術
- HTML
- CSS
- JavaScript

### ライブラリ
- 外部ライブラリ不使用

### 主な実装要素
- CSS Variables によるテーマ切り替え
- JavaScript による現在時刻更新
- JavaScript による時間帯テーマの切り替え
- JavaScript によるボトル充填率計算
- CSS Animation による抽出・ミスト・液体・花びら演出
- Media Query による PC / スマホ切り替え

## 時間帯テーマについて

24時間それぞれに対して、以下の情報を定義しています。

- テーマ名
- 英語テーマ名
- 色のイメージ
- 花の名前
- 花言葉
- Scent Profile
- ボトル用タイトル
- テーマカラー（primary / secondary / accent）
- 花モチーフ描画用の形状設定

これにより、**0時から23時までそれぞれ異なる香りと色彩の表情** を楽しめます。

## 使い方

1. `index.html` をブラウザで開く
2. 自動的に現在時刻が表示される
3. 時間帯に応じて、花・色・香り説明が切り替わる
4. ボトルは10分ごとに満ち、満了時に花びら演出が入る

## カスタマイズしやすいポイント

### テーマ変更
JavaScript内の `themes` 配列を書き換えることで、各時間帯の以下を変更できます。

- 花名
- 花言葉
- 香り説明
- テーマ色
- 花モチーフの形状

### デザイン変更
CSSの以下を中心に調整すると印象を変えやすいです。

- `:root` の色変数
- `.screen`
- `.card.floral-card`
- `.bottle-body`
- `.screen-petal`
- 各種 `@keyframes`

### 演出速度変更
JavaScript側の充填率計算を変更することで、1本あたりの充填時間を変えられます。
